# AITRPG API

NestJS backend for AITRPG.

## Responsibilities

- email-code authentication
- campaign and character APIs
- room and story ledger APIs
- co-DM suggestion endpoints
- afterplay media job submission

## Commands

```bash
pnpm --filter api test
pnpm --filter api test:e2e
pnpm --filter api build
pnpm --filter api start:dev
```
