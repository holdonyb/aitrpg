# ATRPG UI Prototype

## Surface Map

1. Landing and login
2. Campaign creation
3. Character forge
4. Live session room
5. Afterplay studio
6. Admin system page

## Wireframe Notes

### Landing and login

- top hero with product pitch, Chinese-first copy
- email login panel with code request and verify states
- secondary panel showing latest campaigns after login

### Campaign creation

- left column: world template, tone, party setup
- right column: room defaults, AI assistant toggles, media style

### Character forge

- structured character sheet form
- portrait preview card
- portrait generation job state

### Live session room

- center timeline for room messages
- right rail for AI co-DM suggestions
- left rail for participants, characters, and current scene summary
- footer composer with role context and submit action

### Afterplay studio

- tabs for illustration, novel, video
- each tab lists prior jobs, current status, and result assets

### Admin system page

- provider models
- queue health
- storage health
- latest failed jobs
